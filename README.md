# 🍲 Recipe Web Page

## 📌 Project Overview

This project is a simple and informative **Recipe Web Page** built using HTML.
It displays detailed cooking instructions and ingredients for different recipes in a clean and structured format.

The main goal of this project is to practice HTML structure and content organization while creating a real-world useful webpage.

---

## ✨ Features

### 📖 Recipe Details Included

* Recipe Name
* Ingredients List
* Step-by-Step Cooking Instructions
* Preparation Time
* Cooking Time
* Serving Information

### 🧾 Structured Layout

* Proper use of headings
* Ordered and unordered lists
* Clean content organization

---

* HTML5
  *(CSS and JavaScript will be added in future updates to improve styling and interactivity.)*
